# NodeJS Challenge

Desenvolvimento de uma API simples utilizando Node.js, Express, Sequelize e Sqlite.
### Start
    $ yarn install
    $ yarn dev

### Principais Rotas
    get('/api/:username')
	get('/api/repositories/:username)
	get('/api/followers/:username')
	get('/api/followings/:username')
	
	Exemplo de username cadastrado: 'Lucas'
